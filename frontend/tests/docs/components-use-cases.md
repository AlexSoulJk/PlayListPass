# Сценарии тестирования компонентов авторизации

## AuthInput
- Показывает текст ошибки под полем.
- Устанавливает `aria-invalid=true` при наличии ошибки.
- Поддерживает переключение видимости пароля через кнопку с доступными `aria-label`.

## AuthCard
- Стабильно рендерит дочерний контент.
- Используется как базовый контейнер для страниц auth-раздела.
